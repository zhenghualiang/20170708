# 怎样插入图片？
* 先将图片上传。方法有二：
1. 在桌面端将图片复制到相应的文件夹里，再通过change确认[commit]
2. 在网页端upload file添加图片。
* 在.md文件里面插入图片

命令格式为`![插入图片](/image/my-profile.jpg)`
